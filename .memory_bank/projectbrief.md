# Бриф Проекта

## 1. Обзор

**Medieval Hexagon Map Editor** — это современный веб-редактор для создания и редактирования гексагональных карт, реализованный с использованием Next.js 16, React 19, TypeScript и Three.js. Проект представляет собой полнофункциональный 3D редактор карт с поддержкой многоуровневой системы размещения тайлов (до 5 уровней высоты).

Редактор позволяет создавать, редактировать и сохранять пользовательские карты с использованием реальных 3D моделей, современным пользовательским интерфейсом на базе Shadcn/ui и продвинутыми функциями редактирования (копирование/вставка, множественное выделение, undo/redo).

## 2. Ключевые Цели

1. **3D Редактирование**: Полнофункциональный 3D редактор карт с использованием Three.js и реальных 3D моделей
2. **Многоуровневая система**: Поддержка до 5 уровней высоты для создания сложных ландшафтов
3. **Современный стек**: Next.js 16 + React 19 для быстрой разработки и отличного UX
4. **LLM-совместимость**: Использование осевых координат (q, r) для совместимости с LLM-генерацией карт
5. **Производительность**: Оптимизированная работа с большими картами и множеством тайлов

## 3. Границы Проекта (Scope)

### Входит в объем (In-Scope)

- 3D редактор гексагональных карт
- Многоуровневая система размещения тайлов (0-4 уровня высоты)
- Размещение различных типов местности и зданий
- Копирование и вставка тайлов (Ctrl+C, Ctrl+V)
- Множественное выделение тайлов (Ctrl+ЛКМ)
- Система undo/redo для операций редактирования (Ctrl+Z, Ctrl+Y)
- Сохранение и загрузка карт (JSON формат с версионированием)
- Использование реальных 3D моделей из папки assets/
- Современный UI на базе Shadcn/ui компонентов

### Не входит в объем (Out-of-Scope)

- Игровая логика (движение юнитов, боевая система и т.д.)
- Мультиплеер/совместное редактирование
- Облачные сервисы для сохранений
- Мобильная версия (только десктоп веб)
- Редактор кампаний (только редактор карт)
- Система достижений и статистики

## 4. Критерии Успеха

1. Редактор полностью функционален в браузере без дополнительных плагинов
2. Редактор позволяет создавать и сохранять пользовательские карты
3. 3D рендеринг работает с приемлемой производительностью (30+ FPS)
4. Код покрыт тестами для критических компонентов
5. Документация актуальна и синхронизирована с кодом (Memory Bank)

## 5. Технологические Ограничения

- **Браузерное приложение**: Работает в современных браузерах без серверной части
- **Локальные данные**: Все сохранения в LocalStorage браузера
- **Статические assets**: Модели и текстуры хранятся локально в папке assets/
- **Next.js**: Использование Next.js 16 с App Router и React Server Components где применимо
